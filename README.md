# TOPSIS Command Line Tool (Python)

This repository contains a command-line implementation of the **TOPSIS (Technique for Order Preference by Similarity to Ideal Solution)** method in Python.

The program reads a CSV file, applies TOPSIS using user-defined weights and impacts, and produces a ranked output file.

---

## Features

- Command-line based execution
- Complete input validation
- Handles file not found and invalid formats
- Ensures numeric criteria values
- Generates TOPSIS score and rank
- Outputs result as a CSV file

---

- Python 3.8+
- pandas
- numpy
